Before running anything:
  1. Run pip install requirements.txt to install all necessary dependencies
     
How to run Telecom Churn:
  1. Run the starter file to run the models and generate keras_gradients.json and model_leaderboard_telco.json, which will be placed into outputs folder.
	2. Run graph_generator.py to generate png of gradient over time and AUC and accuracy of all models, which will be placed into outputs folder.

How to run Bitcoin Forecasting:
	1. Run jupyter notebook blocks one at a time or all at once, graphs will all be shown in the notebook.
